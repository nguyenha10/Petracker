COLOUR PALETTE

While users will be able to select from a number of different themes, the default colour scheme is comprised of two contrasting tones. The green and purple colours will form the main UI, while a light grey will add some contrast for primary elements. White has been selected for the large, informative textual elements and the purple will be used for more secondary text.

FONT CHOICE

Two sans-serif fonts will be used. This choice was fairly evident, as serif fonts do not fit into the simple, minimal and informative aesthetic 4C aims to achieve. Noto Sans will serve as the main font-family, used for paragraphs and smaller text elements. Roboto is a very strong bold font, and will be used for larger text elements, such as the temperature or rainfall weather components.
